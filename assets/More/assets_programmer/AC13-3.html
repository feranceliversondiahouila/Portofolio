AC13.03 : Traduire un algorithme, dans un langage et pour un environnement donné

Passer d'un algorithme exprimé en pseudo-code ou en langage naturel à une implémentation fonctionnelle dans un langage de programmation adapté.

📚 Modules associés

Les ressources (modules) qui travaillent cette compétence :

  • R110 — Programmation Python — implémentation d'algorithmes
  • R111 — Algorithmique — pseudo-code, organigrammes, complexité
  • R115 — Langages de script réseau — Bash, Python pour l'automatisation

🧠 Notions clés à maîtriser

  • Pseudo-code et organigrammes
  • Structures de données : listes, piles, files, dictionnaires
  • Algorithmes de tri (bulles, insertion, rapide) et de recherche
  • Récursivité : principe et exemples
  • Complexité algorithmique : O(n), O(n²)
  • Traduction pseudo-code → Python
  • Scripts Bash pour l'automatisation de tâches réseau
  • Paramètres de ligne de commande (argparse, sys.argv)

🛠️ SAÉ liées

Situations d'apprentissage et d'évaluation mobilisant cette compétence :

  • SAE13 — Implémentation d'un outil réseau à partir d'un cahier des charges
  • SAE14 — Script d'automatisation de sauvegarde/configuration
