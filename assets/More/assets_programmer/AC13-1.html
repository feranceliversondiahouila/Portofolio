AC13.01 : Utiliser un système informatique et ses outils

Être à l'aise avec l'environnement de travail informatique : système de fichiers, ligne de commande, outils de productivité et éditeurs de code.

📚 Modules associés

Les ressources (modules) qui travaillent cette compétence :

  • R108 — Introduction aux systèmes informatiques — architecture matérielle et logicielle
  • R109 — Systèmes d'exploitation — Linux CLI et Windows
  • R110 — Outils du développeur — éditeurs (VS Code, Vim), terminal, gestionnaires de paquets

🧠 Notions clés à maîtriser

  • Architecture d'un ordinateur (CPU, RAM, stockage)
  • Système de fichiers Linux et Windows
  • Commandes shell : ls, cp, mv, grep, find, pipe
  • Gestionnaires de paquets : apt, pip, npm
  • Éditeurs de code : VS Code, raccourcis clavier
  • Contrôle de version Git : init, add, commit, push, pull
  • Variables d'environnement et configuration du shell (.bashrc)
