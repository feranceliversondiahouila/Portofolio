AC13.05 : Choisir les mécanismes de gestion de données adaptés au développement de l'outil et argumenter ses choix

Sélectionner et utiliser la solution de stockage et de gestion de données la plus adaptée (base de données relationnelle, fichiers, API) selon les besoins de l'application.

📚 Modules associés

Les ressources (modules) qui travaillent cette compétence :

  • R113 — Bases de données — modèle relationnel, SQL
  • R118 — Gestion de données — fichiers CSV/JSON, bases NoSQL
  • R119 — Conception de bases de données — MCD, MLD, normalisation

🧠 Notions clés à maîtriser

  • Modèle relationnel : tables, clés primaires/étrangères
  • SQL : SELECT, INSERT, UPDATE, DELETE, JOIN
  • Conception d'un MCD (Modèle Conceptuel de Données)
  • SQLite pour les petites applications Python
  • MySQL / MariaDB : installation et administration de base
  • Fichiers JSON et CSV comme source de données
  • Introduction aux bases NoSQL (MongoDB)
  • ORM : SQLAlchemy (Python) — requêtes sans SQL brut

  • SAE14 — Analyse et stockage de logs réseau en base de données
