AC13.04 : Connaître l'architecture et les technologies d'un site Web

Comprendre comment fonctionne le Web, du protocole HTTP aux architectures client-serveur, et être capable de créer des pages web statiques et dynamiques basiques.

📚 Modules associés

Les ressources (modules) qui travaillent cette compétence :

  • R112 — Développement Web — HTML5, CSS3, JavaScript
  • R116 — Architecture Web — protocole HTTP/HTTPS, serveurs web (Apache, Nginx)
  • R117 — Frameworks et CMS — introduction à Flask/Django, WordPress

🧠 Notions clés à maîtriser

  • HTML5 : structure d'une page, balises sémantiques
  • CSS3 : sélecteurs, box model, flexbox, grid
  • JavaScript : DOM, événements, fetch API
  • Protocole HTTP : méthodes GET/POST, codes de statut (200, 404, 500)
  • HTTPS et certificats TLS/SSL
  • Architecture MVC (Modèle-Vue-Contrôleur)
  • Serveur web Apache/Nginx : configuration de base
  • API REST : requêtes JSON, endpoints

Situations d'apprentissage et d'évaluation mobilisant cette compétence :

  • SAE13 — Création d'un site web de présentation de projet
  • SAE15 — Déploiement d'une application web sur un serveur Linux
